## 项目概述

面向 DevOps 团队的一站式监控解决方案，将来自多个数据源的指标整合为统一视图。

## 核心功能

- **统一指标**：接入 Prometheus、CloudWatch 和 Datadog
- **智能告警**：基于 ML 的异常检测和分级告警策略
- **事件时间线**：可视化事件追踪和根因分析
- **Runbooks**：告警触发自动化修复剧本

## 技术栈

- **前端**：Vue 3 + Composition API + ECharts 可视化
- **后端**：Go + Gin 框架 + gRPC 微服务
- **数据采集**：Prometheus + Grafana Agent
- **部署**：Kubernetes + Helm Charts
