## 项目概述

企业级设计系统，提供统一的组件库和设计规范，确保多产品间的视觉一致性。

## 核心功能

- **组件库**：50+ 生产级 React 组件（按钮、表单、模态框等）
- **设计令牌**：统一的色彩、字体、间距、阴影等设计变量
- **Storybook 文档**：交互式组件文档和用例展示
- **主题系统**：支持亮色/暗色模式和自定义主题

## 技术栈

- **组件开发**：React + TypeScript + Emotion CSS-in-JS
- **文档**：Storybook 7 + MDX
- **测试**：Jest + React Testing Library + Chromatic 视觉回归测试
- **发布**：npm + Changesets 版本管理
