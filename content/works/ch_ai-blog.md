## 项目概览

AI 博客系统是一个基于 Next.js、Tailwind CSS 和 MDX 构建的现代复古风格静态博客。专为注重简洁设计与强大功能的开发者和内容创作者打造。

## 核心特性

- **静态站点生成 (SSG)** - 预渲染 HTML 带来极快的页面加载速度
- **MDX 支持** - 结合 Markdown 语法和 React 组件撰写富文本内容
- **国际化 (i18n)** - 内置中英文双语支持
- **新粗野主义设计 (Neo-Brutalism)** - 具有硬边界和实心阴影的现代复古美学
- **暗色模式** - 流畅的深浅色主题切换
- **响应式设计** - 完美适配各类设备屏幕
- **免费托管** - 零成本部署至 GitHub Pages 或 Vercel

## 技术栈

- **框架**: Next.js 16 (App Router)
- **UI**: React 19 + Tailwind CSS 4
- **内容**: 基于 `next-mdx-remote` 的 MDX 解析
- **元数据**: 使用 `gray-matter` 解析 YAML frontmatter
- **主题**: 集成 `next-themes` 实现暗黑模式
